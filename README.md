# 🎨 Random – Visual Identity & Web3 Experience Portfolio

RANDOOM is a cinematic brand identity designed for a high-end NFT and logo designer. This project bridges the gap between digital asset creation and professional web presentation, focusing on a dark, atmospheric aesthetic that caters to the modern crypto and NFT market

---

## ✨ Features

- 🖌️ **Visual Identity** – Custom graphics, typography, color systems
- 🌐 **Web3 Ready** – Wallet connection, NFT gallery, on‑chain data (planned)
- ⚡ **Smooth Animations** – Framer Motion for page transitions and micro‑interactions
- 📱 **Fully Responsive** – React-Bootstrap grid and custom CSS
- 🎨 **Modular Components** – Reusable, maintainable code

---

## 🧰 Tech Stack

| Layer      | Technology                    |
| ---------- | ----------------------------- |
| Frontend   | React 18 + Vite               |
| UI Library | React-Bootstrap + React Icons |
| Animations | Framer Motion + React Typed   |
| Styling    | CSS (custom + Bootstrap)      |
| Hosting    | Vercel                        |
